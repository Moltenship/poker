## Key Finding: convex-test moduleCache path format
The modules map passed to convexTest() must use keys with .js extensions
and starting with ./ (not ../) for correct resolution by convex-test's
moduleCache function. The regex /\.[^.]+$/ in moduleCache strips file
extensions, but "../rooms" gets mangled to "." because the regex matches
from the second dot in "../". The correct pattern:
  "./_generated/api.js": () => Promise.resolve({ default: api }),
  "./rooms.js": () => Promise.resolve(rooms),
